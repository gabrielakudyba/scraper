    How it works

    • Optimized adapters for clocate.com and 10times.com.
    • Clocate: parses listing cards, JSON-LD, past editions, venue, category, frequency.
    • 10times: uses a real browser (Playwright) to bypass Cloudflare — enable Force browser mode.
    • If 10times shows a verification page, restart with HEADLESS=false npm start and complete it once.

    Example URLs: https://www.clocate.com/conferences-in+germany/Y28tZGU=/ or https://10times.com/events
